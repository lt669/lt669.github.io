Location Tracking

This video demonstrates the location tracking functionality of the user interface, using the c74 object in Max. A user can draw themselves a path on the iPad when placed within the Virtual Singing Studio (VSS). Then, within the digitally designed virtual space, they will be moved around the path, allowing them to hear themsevels as though they are walking around the room. Referenced in Section Implementation: Software: Location Tracking: User Interface

Iteration 1: Multiple RIR Interpolation

This video demonstrates the functionality of the interpolation software designed for the original Max patch (Iteration 1) By selecting a position on the user interface (in this case done on a computer instead of the iPad), the software will calculate which RIR files can closest to the users desired location and then interpolate between the nearest 4 RIRs to simulate an approximation of the room acoustics. Referenced in Section Implementation: Software: Mobility Implementation: Iteration 1
