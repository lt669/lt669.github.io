Max/Javascripts

The following table provides links to all of the Javascript files used in the Max/MSP direct room impulse response rendering system along along with a short description and in which section of the document they are referenced.

File Name Description Section Referenced
gridLoading.js Converts the user interface coordinates into an appropriate file name Software:Max
ifFirst.js Prevents the 3rd convolution patch from fading in when the first patch has faded in for the first time. Software:Max
loadFiles.js Concatenates the input file number with the appropriate file path to find the required RIR file when using iPad interface. Software:Max
loadFilesLogic.js Calculates a file number (1 - 240) based on the coordinates from the user interface Software:Max
loadFilesTest1.js Concatenates the input file number with the appropriate file path to find the required RIR file for user test #1. Software:Max
storeCood.js Stores the coordinates of the user input, used to draw the user posistion dot on the user interface Software:Max
storeLocations.js Stores the list of files names that still need to be loaded into the convolution patches. Software:Max
switcher.js Routes the input signal to revolving convolution patches. Software:Max
timerLogic.js Timer: Outputs a signal ever 2.5 seconds Software:Max
wrapper.js Keeps the input value between 0 - 360 Software:Max
