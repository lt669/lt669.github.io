Matlab Scripts

This page contains all the Matlab files used in the project as referenced in the document

File Name Description Section Referenced
calibrateAll.m Used in an attempt to accurately calibrat real and synthetic RIRs Implementation: Odeon
renameSortFiles.m Extracts and sorts the RIR files to produce 4 new RIR grids Implementation: Odeon
RIRtrim.m Trims RIR files. Used to correct error caused by system latency Software:Max
deconvolve.m Deconvolved captured sinusoidal sweep (Adapted from University of York Msc Music Technology Webpage) Implementation: Real RIR Measurements
generatesweep.m Produces a sinusoidal sweep for RIR recording (Adapted from University of York Msc Music Technology Webpage) Implementation: Real RIR Measurements
