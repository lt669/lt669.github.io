Location Tracking

This video demonstrates the location tracking functionality of the user interface, using the c74 object in Max. A user can draw themselves a path on the iPad when placed within the Virtual Singing Studio (VSS). Then, within the digitally designed virtual space, they will be moved around the path, allowing them to hear themsevels as though they are walking around the room.
